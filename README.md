# Projeto Reconhecimento de Imagens: webapp-fire

# # Modelo de IA com Teachable Machine


Este projeto é parte de um trabalho acadêmico voltado para o reconhecimento de imagens utilizando **Teachable Machine**, uma ferramenta de aprendizado de máquina acessível e prática.

O modelo foi treinado com um conjunto balanceado de **1.780 imagens**, sendo:
- **890 imagens de áreas em incêndio** (*fire*),
- **890 imagens de áreas preservadas** (*nofire*).

O objetivo é oferecer uma aplicação web capaz de identificar, a partir de imagens enviadas pelo usuário, se há ocorrência de incêndio florestal ou se a área está preservada.  
O sistema foi desenvolvido em **JavaScript**, com suporte do **TensorFlow.js**, e possui interface simples e responsiva para facilitar o uso em diferentes dispositivos.
